Getting Started With Hash Tables

What is a Hash Table?

A hash table is a data structure that maps keys to values. You can think of it as a dictionary for those familiar with python programming language, as dictionaries are implemented using hash tables. A hash table is like an array of fixed size(decided when creating the table). Then the nodes(items in the table) contain keys and values. A value is stored based on the hash value index, derived from the key, providing a very fast access time. We compute the index of a value using a chosen hash function. We will look at hash functions later on.
